# CutPPaper 接线说明（简洁版）

> **MCU：** STM32F103C8T6 · **串口：** 115200 8N1 · **固件：** `firmware/CutPPaper.uvprojx`  
> **默认配置：** 外接 2.4" ST7789V 屏（`BOARD_LCD_MODE=1`）  
> **一页速查：** [`wiring-table.md`](wiring-table.md)

---

## 1. 引脚总表

| 引脚 | 接什么 | 说明 |
|------|--------|------|
| **PA0** | 继电器 IN1 | 电机缩回 K1 |
| **PA1** | 继电器 IN2 | 电机伸出 K2 |
| **PA2** | 继电器 IN3 | 继续键脉冲 K3 |
| **PA3** | 继电器 IN4 | 原点键脉冲 K4 |
| **PA4** | LED → 330Ω → GND | 缩回指示灯 |
| **PA5** | LED → 330Ω → GND | 伸出指示灯 |
| **PA6** | — | 未用（有屏时） |
| **PA7** | — | 未用（有屏时） |
| **PA8** | 槽型光电 DO | 遮挡 = 压纸中 |
| **PA9** | USB-TTL **RX** | MCU 发送 TX |
| **PA10** | USB-TTL **TX** | MCU 接收 RX |
| **PA13** | ST-LINK SWDIO | 烧录 |
| **PA14** | ST-LINK SWCLK | 烧录 |
| **PB8** | 按钮 → GND | 现场缩/伸切换 |
| **PB9** | 通信灯 D3 | 有屏板板载 LED |
| **PB10** | LCD **BLC** | 背光，高电平亮 |
| **PB11** | LCD **RES** | 屏复位 |
| **PB12** | LCD **CS** | 片选 |
| **PB13** | LCD **SCL** | SPI2 时钟 |
| **PB14** | LCD **DC** | 数据/命令 |
| **PB15** | LCD **SDA** | SPI2 数据 |

> 通信灯是 **PB9**，不是 PA9。

## 2. 分项接线

### 2.1 电脑通信（USB-TTL）

| USB-TTL | STM32 |
|---------|-------|
| TX | PA10 |
| RX | PA9 |
| GND | GND |
| VCC | **不接 MCU**（模块拨到 3.3V） |

### 2.2 烧录（ST-LINK）

| ST-LINK | STM32 |
|---------|-------|
| SWDIO | PA13 |
| SWCLK | PA14 |
| GND | GND |
| 3.3V | 可选 |

### 2.3 四路继电器模块

**控制端**（跳线 **H** = 高电平触发）：

| 继电器 | STM32 | 功能 |
|--------|-------|------|
| DC+ | 12V（勿接 24V） | 模块供电 |
| DC− | GND | |
| IN1 | PA0 | K1 缩回 |
| IN2 | PA1 | K2 伸出 |
| IN3 | PA2 | K3 继续 |
| IN4 | PA3 | K4 原点 |

**K1/K2 — 24V 伸缩杆（H 桥）：**

```
24V+ ── NO1、NO2（短接）
GND  ── NC1、NC2（短接）
电机线 A ── COM1
电机线 B ── COM2
```

| PA0 | PA1 | 动作 |
|-----|-----|------|
| 1 | 0 | 缩回 |
| 0 | 1 | 伸出 |
| 0 | 0 | 停止 |

**K3/K4 — 切纸机按钮（干触点）：**

```
继续按钮 ── COM3 / NO3
原点按钮 ── COM4 / NO4
（NC3、NC4 不接）
```

### 2.4 槽型光电（压纸检测）

| 光电 | 接法 |
|------|------|
| VCC | 3.3V |
| GND | GND |
| DO | PA8 |
| AO | 不接 |

遮挡 = 压纸中（`ROD:HOME`）· 无遮挡 = 未压纸（`ROD:AWAY`）

### 2.5 现场按键

```
PB8 ── 按钮 ── GND
```

按一次缩回 3 秒，再按一次伸出 3 秒，循环。

### 2.6 外接 2.4 寸 SPI 屏（ST7789V · 8 针）

模块针脚：**GND · VCC · SCL · SDA · RES · DC · CS · BLC**  
PCB 跳线 J1 选 **3.3V Short**。

| 屏 | STM32 |
|----|-------|
| GND | GND |
| VCC | 3.3V |
| SCL | PB13 |
| SDA | PB15 |
| RES | PB11 |
| DC | PB14 |
| CS | PB12 |
| BLC | PB10 |

默认竖屏 240×320（`LCD_USE_HORIZONTAL=0`）；横屏 320×240 改为 `2`。

## 3. 电源与共地

```
220V → 24V 电源 → 24V+/GND → 继电器触点（电机）
                           ↘
                 DC-DC → 3.3V → STM32、光电、LCD
```

**必须共地：** 24V− = STM32 GND = USB-TTL GND = 继电器 DC− = ST-LINK GND  

**禁止：** 24V 正极接 MCU 引脚 · USB-TTL 用 5V 档 · 继电器 DC+ 接 24V

---

## 4. 无屏模式（`BOARD_LCD_MODE=2`）

不接 LCD，下列引脚改作它用，**其余与上文相同**：

| 引脚 | 功能 |
|------|------|
| PA6 | 通信灯 D3 |
| PA7 | 切换键 → GND |
| PB10~PB15 | 未用 |

有屏模式（默认）：通信灯 **PB9**，切换键 **PB8**。

---

## 5. 软件 ↔ 硬件

| 流程步骤 | 串口/软件 | 硬件 |
|----------|-----------|------|
| 缩回 | `RETRACT` | PA0 → K1 |
| 模拟按键 B | `PULSE_A` | PA2 → K3 |
| 发送切割 | Ctrl+P | Cutting Master 4 |
| 伸出 | `EXTEND` | PA1 → K2 |
| 模拟按键 A | `PULSE_B` | PA3 → K4 |
| 急停 | `ESTOP` | 电机停 + 继电器全关 |

---

## 6. 上电前检查

1. 所有 GND 已共地  
2. 串口 TX/RX 已交叉，波特率 115200  
3. H 桥：NO→24V+，NC→GND，COM→电机  
4. 光电 DO→PA8，LCD/光电/继电器控制线均为 3.3V 逻辑  
5. 先不接 24V，通电测 K1/K2 吸合与串口 `PING` → `OK:PONG`  
6. 再空载试转，最后接负载
