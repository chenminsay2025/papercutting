# CutPPaper 接线总表

> 固件：`firmware/CutPPaper.uvprojx` · 主控：STM32F103C8T6 · 继电器：四路 U4 · 串口：115200 8N1  
> **默认硬件：nologo STM32F103C8T6/C6T6 + 0.96" TFT 一体板**（板载 LCD / Flash，外设引脚见 §六、§六点六）  
> 图例：[`wiring-diagram.html`](wiring-diagram.html) · 详述：[`wiring.md`](wiring.md)

**电机 H 桥：** NO1–NO2 拱桥 → 24V+，NC1–NC2 拱桥 → GND，COM1/COM2 → 伸缩杆两根线。  
**继电器触发：** 跳线 S1~S4 插 **H**（高电平触发，3.3V 吸合）。  
**压纸检测：** PA8 槽型光电遮挡 = 压纸中（已缩回）；未遮挡 = 未压纸（已伸出）。

---

## 一、通信与烧录

**01**　USB-TTL TX → STM32 PA10（USART1_RX）  
　功能：电脑发、MCU 收  
　要求：与 RX 交叉接线；模块 3.3V TTL；115200 8N1  

**02**　USB-TTL RX → STM32 PA9（USART1_TX）  
　功能：MCU 发、电脑收  
　要求：与 TX 交叉接线；VCC 勿接 MCU  

**03**　USB-TTL GND → STM32 GND  
　功能：串口共地  
　要求：必接；与 24V 负极、继电器 DC− 共地  

**04**　USB-TTL VCC → （不接 MCU）  
　功能：模块自供电  
　要求：拨 3.3V 档；勿给 STM32 反向供电  

**05**　ST-LINK SWDIO → STM32 PA13  
　功能：SWD 烧录  
　要求：可与 USB-TTL 同时插；线色建议绿  

**06**　ST-LINK SWCLK → STM32 PA14  
　功能：SWD 烧录  
　要求：BOOT0 接 GND；线色建议蓝  

**07**　ST-LINK GND → STM32 GND  
　功能：烧录共地  
　要求：必接  

**08**　ST-LINK 3.3V → STM32 3.3V  
　功能：烧录供电  
　要求：可选；板子已有 3.3V 时可不接  

---

## 二、电源与共地

**09**　220V AC → 24V 开关电源  
　功能：主电源  
　要求：电流 ≥ 电机额定，建议留 30% 余量  

**10**　24V+ → NO1 与 NO2（短接拱桥）  
　功能：电机 H 桥正极  
　要求：24V 只接触点侧；勿接继电器 DC+  

**11**　24V−（GND）→ NC1 与 NC2（短接拱桥）  
　功能：电机 H 桥负极  
　要求：与 STM32、TTL、ST-LINK、继电器 DC− 共地  

**12**　24V− → 系统 GND 总线  
　功能：共地参考  
　要求：所有 GND 汇一点；24V+ 禁止接 MCU 引脚  

**13**　12V 电源+ → 继电器 DC+  
　功能：继电器线圈/光耦  
　要求：12V 模块接 12V，5V 模块接 5V；勿接 24V  

**14**　系统 GND → 继电器 DC−  
　功能：继电器控制回路地  
　要求：与 24V−、STM32 GND 相连  

**15**　3.3V 电源 → STM32 3.3V  
　功能：MCU 供电  
　要求：禁止 24V 直连；可与 ST-LINK 3.3V 二选一  

---

## 三、继电器控制线

**16**　跳线 S1~S4 → 插 H 位  
　功能：高电平触发  
　要求：与当前固件一致；插 L 须改固件或改回 H  

**17**　STM32 PA0 → 继电器 IN1（K1）  
　功能：伸缩杆缩回  
　要求：缩回时仅 K1 吸合（PA0=1，PA1=0）  

**18**　STM32 PA1 → 继电器 IN2（K2）  
　功能：伸缩杆伸出  
　要求：伸出时仅 K2 吸合（PA0=0，PA1=1）；换向前先停 80ms  

**19**　STM32 PA2 → 继电器 IN3（K3）  
　功能：脉冲「继续」  
　要求：软件 PULSE_A；默认约 200ms  

**20**　STM32 PA3 → 继电器 IN4（K4）  
　功能：脉冲「原点」  
　要求：软件 PULSE_B；默认约 200ms  

---

## 四、伸缩杆电机（K1/K2 · H 桥）

**21**　NO1 ↔ NO2 短接  
　功能：24V+ 分配到 K1/K2  
　要求：端子上用短导线连在一起  

**22**　NC1 ↔ NC2 短接  
　功能：GND 分配到 K1/K2  
　要求：与系统 GND、24V− 共地  

**23**　伸缩杆线 A → COM1  
　功能：电机端 A  
　要求：负载接 COM；勿把 NC/NO 跨并到同一电机线  

**24**　伸缩杆线 B → COM2  
　功能：电机端 B  
　要求：方向反了则对调 COM1/COM2 两根线  

**25**　NC2 → （悬空）  
　功能：未使用  
　要求：不接；旧方案误接会导致停止仍通电  

**26**　停止状态（PA0=0，PA1=0）  
　功能：K1/K2 释放，电机无电压  
　要求：COM1、COM2 经 NC 接 GND  

**PA0 / PA1 与电机：**

- 0 / 0 → 停止（COM1=GND，COM2=GND）
- 1 / 0 → 缩回（COM1=+24V，COM2=GND）
- 0 / 1 → 伸出（COM1=GND，COM2=+24V）
- 1 / 1 → 停止/刹车（COM1=+24V，COM2=+24V）

任意 PA0/PA1 组合不会把 24V+ 与 GND 直接短接。

## 五、切纸机按钮并联（K3/K4）

**27**　切纸机「继续」按钮一端 → COM3  
　功能：并联继续键  
　要求：须低压干触点；220V 须光耦隔离  

**28**　切纸机「继续」按钮另一端 → NO3  
　功能：并联继续键  
　要求：K3 吸合时 COM3–NO3 闭合，模拟按一下  

**29**　NC3 → （悬空）  
　功能：未使用  
　要求：不接  

**30**　切纸机「原点」按钮一端 → COM4  
　功能：并联原点键  
　要求：万用表确认干触点后再并  

**31**　切纸机「原点」按钮另一端 → NO4  
　功能：并联原点键  
　要求：K4 吸合时 COM4–NO4 闭合，模拟按一下  

**32**　NC4 → （悬空）  
　功能：未使用  
　要求：不接

## 六、现场按键与指示灯

**33**　STM32 **PB8** → 切换按钮 → GND（nologo 一体板；**PA7=LCD 复位 RES**）  
　功能：现场手动伸缩  
　要求：常开按钮；按一次缩回 3s，再按伸出 3s，交替；运行中再按无效  

**34**　（nologo 板）PA6 = LCD 背光，**勿接 D3 串口 LED**  
　功能：板载 0.96" 屏背光（低电平亮）  
　要求：外设 LED 接 PA4/PA5；**串口状态灯改 PB9**  

**35**　STM32 PA4 → D1 缩回 LED → 330Ω → GND  
　功能：缩回指示  
　要求：缩回时常亮  

**36**　STM32 PA5 → D2 伸出 LED → 330Ω → GND  
　功能：伸出指示  
　要求：伸出时常亮  

**37**　STM32 **PB9** → D3 串口 LED → 330Ω → GND（**nologo 一体板**；标准板仍用 **PA6**）  
　功能：通信状态  
　要求：快闪=未连接；慢呼吸=已连接

## 六点五、槽型光电（压纸检测）

**45**　槽型光电 VCC → STM32 3.3V  
　功能：传感器供电  
　要求：模块 3.3~5V  

**46**　槽型光电 GND → 系统 GND  
　功能：传感器共地  
　要求：与 STM32、继电器共地  

**47**　槽型光电 DO → STM32 PA8  
　功能：压纸检测（缩回终点位置）  
　要求：遮挡=压纸中/已缩回（`ROD:HOME`）；未遮挡=未压纸/已伸出（`ROD:AWAY`）；缩回中刚遮挡时自动停电机  

**48**　槽型光电 AO → （悬空）  
　功能：未使用  
　要求：本模块 AO 不起作用

## 六点六、板载 0.96" LCD（nologo 一体板 · 已焊死）

**49**　PB0 → LCD DC（数据/命令）  
**50**　PB1 → LCD CS（片选）  
**51**　PB10 → LCD SCL（时钟）  
**52**　PB11 → LCD SDA/MOSI（数据）  
**53**　PA7 → LCD RES（复位）  
**54**　PA6 → LCD 背光 BLK（低电平亮，**勿接 D3**）  
**55**　PB12~PB15 → 板载 W25Q64 Flash（SPI2，**勿占用**）  

LCD 界面（固件）：**切纸机状态** · **压纸** 压纸中/未压纸 · **USB** 已连接/未连接 · **电机** 停止/缩回/伸出  
引脚定义与 `屏幕显示_源码示例/HARDWARE/LCD/lcd_init.h` 一致。

## 七、软件（无硬件线）

**38**　CutPPaper → Cutting Master 4  
　功能：发送切割  
　要求：默认 Ctrl+P；须先激活目标窗口  

**39**　CutPPaper → USB-TTL COM 口  
　功能：流程控制  
　要求：设备管理器确认 COM 号；烧录或拔 USB 后须重新连接  

**40**　CutPPaper 串口连接  
　功能：与 MCU 通信  
　要求：CH340 若 RTS 接 NRST，软件已固定 DTR=1、RTS=1，连接后等约 350ms

## 八、软件步骤 ↔ 硬件

- **A 缩回** · 命令 `RETRACT` · PA0=1 PA1=0，仅 K1；PA8 压纸中（`ROD:HOME`）时自动停电机
- **B 继续** · 命令 `PULSE_A` · PA2 脉冲，K3 吸合，COM3/NO3 闭合
- **C 切割** · PC 热键 · Cutting Master 4，无继电器
- **D 伸出** · 命令 `EXTEND` · PA0=0 PA1=1，仅 K2
- **E 原点** · 命令 `PULSE_B` · PA3 脉冲，K4 吸合，COM4/NO4 闭合

**状态显示：** LCD 屏 + CutPPaper 上位机同步显示压纸中/未压纸、连接状态；D3 串口灯接 **PB9**（快闪=未连接，慢呼吸=已连接）。

## 九、线材

- **41** 24V 电机线 · 0.75~1.5 mm² · 尽量短，与信号线分开
- **42** STM32 信号线 · 杜邦线 · PA0~PA3、串口、SWD
- **43** 继电器→切纸机按钮 · 0.5 mm² · 与电机线分开，可选屏蔽
- **44** 共地 · 较粗或星形汇流 · 避免地环路

## 十、上电前检查

1. 所有 GND 共地（24V−、STM32、TTL、ST-LINK、继电器 DC−）
2. 24V 不接 DC+；DC+ 接 12V（或 5V 模块接 5V）
3. 电机：NO1–NO2 接 24V+，NC1–NC2 接 GND，COM1/COM2 接电机；NC2 悬空
4. 串口 TX/RX 交叉，115200；VCC 不接 MCU
5. 跳线 S1~S4 插 H；K3/K4 并机器前确认干触点
6. 槽型光电：VCC→3.3V，GND 共地，DO→PA8，AO 悬空；遮挡时 `ROD_SENSOR` 返回 `ROD:HOME`（压纸中）
7. nologo 板：D3 串口 LED 接 **PB9**（勿用 PA6）；切换键接 **PB8**；PA6/PA7/PB0/PB1/PB10/PB11 为板载 LCD
8. 先不接 24V 测 K1/K2；再空载试转；最后并机器按钮联调

---

## 十一、禁止

- **X1** 24V 接继电器 DC+ → 烧线圈/光耦
- **X2** 24V+ 接 STM32 引脚 → 烧 MCU
- **X3** 5V TTL 直连 STM32 → 烧 MCU
- **X4** COM1=24V、COM2=GND 旧接法 → 停止时电机可能仍通电
- **X5** NC 与 NO 跨继电器并到同一电机线 → 易短路
- **X6** 强电按钮直接并 COM/NO → 须干触点或光耦隔离
