# LCD Font Studio

独立于 CutPPaper 的 **LCD 字模生成器**。用 TTF/TTC（苹方、雅黑等）通过 **超采样 + 缩小 + 锐化 + 16 级 AA** 生成点阵字库。

**半宽规则（与 STM32 一致）**：ASCII 字母、数字、符号在 **左半格**（如 16×16 时为 8px 宽）渲染；中文及全角符号占 **整格**。预览「LCD 模拟」对半宽字只画左半边。

## 安装

```powershell
cd lcd-font-studio
pip install -r requirements.txt
```

## 运行

```powershell
python main.py
```

或双击 `run.bat`（Windows）。

## 功能

| 功能 | 说明 |
|------|------|
| 字体选择 | 任意 `.ttf` / `.ttc` / `.otf`，支持 TTC 索引 |
| 参数调节 | 宽高、超采样倍数、二值化阈值、ASCII/汉字缩小量 |
| 实时预览 | 单字 1-bit 点阵 + 灰度中间图 |
| 批量生成 | 粘贴或从 `.txt` 加载字符集（自动去重） |
| 导出 CutPPaper 包 | `lcd_font_gb16.c` + `.h` + 索引表 + 文本预览 |
| 导出 PNG / 二进制 | 字模总览图、`glyphs.bin` |
| 预设 | 保存/加载 JSON 预设 |

先点 **「锐化」** 或 **「强锐化」** 预设，再微调：

| 参数 | 作用 | 建议 |
|------|------|------|
| **锐化强度 %** | Unsharp 增强量，0=关闭 | 120–200 常用，过高会噪点 |
| **锐化半径** | 边缘检测模糊半径 | 0.5–0.8，越小笔画越利 |
| **锐化阈值** | 低于此灰度差不锐化（保护背景） | 2–4 |
| **高斯模糊** | 缩小前柔化，与锐化配合 | 锐化时可降到 0–0.5 |

流程：超采样渲染 → 缩小 → Gamma → **锐化** → 导出 16 级 AA 字库。

## 导出到 CutPPaper

1. 生成字模 → **导出 CutPPaper 字库包**（默认 **16 级 AA 灰度**，162 字节/字）
2. 或运行：`python export_cutppaper_aa.py`（直接写入 `../firmware/User/`）
3. Keil **Rebuild (F7)** → **Download (F8)**

固件 `lcd_st7735.c` 用 `Lcd_Blend565()` 按灰度混合前景/背景，**LCD 上才有抗锯齿**。

预览窗口左侧 **「LCD 效果」** 用 NEAREST 放大，模拟屏上真实像素（不做平滑糊弄）。

> 英文/数字仍用固件半宽 `lcd_font_ascii16.c`（1-bit）。

## 推荐参数（0.96" ST7735）

先点 **「极平滑」** 预设，再微调：

| 参数 | 建议值 |
|------|--------|
| 宽 × 高 | 18 × 18 |
| 超采样 | **6–8** |
| 缩小算法 | **box**（面积平均，斜线更稳） |
| 高斯模糊 | **1.0–1.4** |
| Gamma | **1.15–1.22** |
| 阈值 | **115–128**（笔画太细可略降） |
| 二值化 | threshold（中文推荐）；floyd 仅细线场景 |

### 关于「锯齿」

ST7735 是 **1-bit 单色屏**，每个像素只能全亮或全灭，斜线/曲线在物理上必然有阶梯感。工具做的是尽量在转成 1-bit 前保留灰度信息；若仍不满意，只能：

- 加大字号（如 20×20，需同步改固件 `LCD_GB16_W/H`）
- 或改用灰度字库（4/16 级 AA，需改 `lcd_st7735.c` 绘制逻辑）

## 目录结构

```
lcd-font-studio/
  main.py              # 入口
  run.bat              # Windows 启动
  requirements.txt
  font_studio/
    core.py            # 渲染核心
    export.py          # 导出
    app.py             # 窗口 UI
```

与 `firmware/tools/gen_lcd_font.py` **无代码依赖**，可单独拷贝本文件夹到其他机器使用。
