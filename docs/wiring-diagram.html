ST-LINK 接线要点

  • 线色建议：SWDIO 绿 · SWCLK 蓝 · GND 黑 · 3.3V 红（可选）
  • 板子已有独立供电时，3.3V 可不接，只接 SWDIO、SWCLK、GND
  • 与 USB-TTL 可同时连接，互不影响

交叉接线记忆法

  • TTL 的 TX → MCU 的 RX (PA10)
  • TTL 的 RX ← MCU 的 TX (PA9)
  • 不要用 5V TTL 直连 STM32！VCC 不要接到 MCU

并联接线示意

  • 找到机器「继续」按钮背后两根线 → 接到继电器 A 的 COM 和 NO
  • 「原点」按钮同样接到继电器 B 的 COM 和 NO
  • 先用万用表确认按钮是干触点（按下通、松开断）

电源规则

  • 12V 只给电机和（如需）降压模块，不要接 STM32 引脚
  • 12V 电源的 负极 = 系统 GND
  • 电机线尽量粗、短；与信号线分开走
